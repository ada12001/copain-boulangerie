Design System

A streamlined foundation for Boulangerie Copain: three core colors, one semantic type scale, and three button treatments. Keep the system simple and reusable.

01 Color

Core colors are limited to black, white, and cream. Text colors are reduced to three roles: default text, text mid, and text light.

02 Typography

Use semantic tags only: H1 through H6, P, and blockquote. Headline copy should be written in sentence case sitewide, with uppercase reserved for the visual H6 utility treatment and button styling.

04 Dividers

05 Section padding

Keep vertical rhythm in the section wrapper, not in one-off heading margins. Most editorial and content sections should inherit the shared site cadence below and only deviate when a full-bleed media treatment truly requires it.

Desktop standard

Use a section shell around 96px 48px 88px for white sections and 88px 48px 96px for denser cream/story sections.

Mobile standard

Collapse to roughly 72px 28px 64px on mobile so headings, dividers, and body copy stay compact without feeling cramped.

Media exceptions

Split layouts, carousels, and full-bleed imagery can set their own internal height, but the section above and below should still return to the shared padding rhythm.

06 Rules

Keep the system disciplined. The goal is fewer exceptions, not more.

Use Semantic Type

Prefer H1 through H6, P, and blockquote over one-off text classes.

Use H6 for Eyebrows

Eyebrows, subtitles, and overlines should all collapse into one uppercase utility style.

Write headlines in sentence case

Page headings, section headings, and editorial statements should use sentence case copy. Let the typography create the drama instead of switching capitalization systems page by page.

Keep Copy in One Body Style

Most supporting copy should use the same paragraph size, color, and line-height.

Reuse the Three Buttons

If a CTA does not fit one of the three approved button styles, the button should be reconsidered.

Use Literal Spacers

When extra vertical space is requested, use a literal spacer element rather than increasing adjacent margins. Use <div class="spacer" aria-hidden="true"></div> or <div class="spacer-40" aria-hidden="true"></div> for 40px, and <div class="spacer-80" aria-hidden="true"></div> for 80px.
